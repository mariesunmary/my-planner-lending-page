# FocusFlow Landing Page

Лендінг сторінка для бакалаврської роботи на тему "Розробка веб-застосунку FocusFlow".

## Про проєкт
FocusFlow — це сучасний застосунок для планування часу, відстеження звичок і фінансів, що допомагає підтримувати баланс між роботою та життям.

## Технології
- **React** (Vite)
- **Sass** (SCSS Modules)
- **GitHub Pages** (Deployment)

## Структура
- `src/components`: Компоненти інтерфейсу (Header, Hero, About, Goals, Methodology, Results, Footer).
- `src/assets`: Стилі та зображення.

## Запуск локально
1. Встановіть залежності:
   ```bash
   npm install
   ```
2. Запустіть сервер розробки:
   ```bash
   npm run dev
   ```

## Деплой
Проєкт налаштований для автоматичного деплою на GitHub Pages.
1. Переконайтеся, що репозиторій створений на GitHub.
2. Змініть `base` у `vite.config.js`, якщо назва репозиторію відрізняється від `my-planner-lending-page`.
3. Запустіть:
   ```bash
   npm run deploy
   ```
   Ця команда автоматично створить білд та завантажить його у гілку `gh-pages`.
